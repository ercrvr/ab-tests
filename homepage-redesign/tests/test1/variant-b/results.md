# Hero Banner Test — Variant B (Minimal)

Minimal hero banner with a dark slate background and an understated bordered CTA button.

## Process

1. Created a full-width hero section with a solid dark background (#0f172a)
2. Used lighter font weight (300) for a refined heading
3. Designed a transparent button with a subtle blue border

## Output

- `index.html` — Full hero section with inline styles
- `styles.css` — Extracted CSS for the hero component

## Assessment

### Strengths
- Clean, modern aesthetic
- Subtle design feels premium
- "Learn More" CTA is less aggressive, may reduce bounce

### Weaknesses
- Lower contrast may reduce immediate click-through
- CTA button is less prominent
