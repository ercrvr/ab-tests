# Hero Banner Test — Variant A (Bold CTA)

Bold hero banner with a purple gradient background and a high-contrast white CTA button.

## Process

1. Created a full-width hero section with a gradient from purple to blue
2. Set heading at 3rem with bold weight for maximum impact
3. Designed a white CTA button contrasting against the gradient

## Output

- `index.html` — Full hero section with inline styles
- `styles.css` — Extracted CSS for the hero component

## Assessment

### Strengths
- Strong visual impact with the gradient
- High contrast CTA button is immediately visible
- Clear call-to-action text "Get Started Free"

### Weaknesses
- Gradient may feel generic
- Bold style might not match all brand aesthetics
