# CTA Button Color Test — Experiment (Orange)

Orange CTA button variant testing whether warmer colors increase click urgency.

## Process

1. Changed button color from green to orange (#f97316)
2. Kept all other properties identical to control (same dimensions, font, radius)

## Output

- `button.html` — Standalone button element with inline styles

## Assessment

### Strengths
- Orange draws more attention against most page backgrounds
- Warmer color may trigger urgency/action
- Same dimensions ensure fair comparison

### Weaknesses
- Orange may clash with certain page color schemes
- Expected CTR: 4.1% (hypothesis)
