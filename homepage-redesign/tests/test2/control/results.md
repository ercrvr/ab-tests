# CTA Button Color Test — Control (Green)

Standard green CTA button used as the baseline for this color test.

## Process

1. Used the existing production green button (#22c55e)
2. Standard padding (14px 36px), 600 font weight, 6px border radius

## Output

- `button.html` — Standalone button element with inline styles

## Assessment

### Strengths
- Standard green conveys "go" / positive action
- Users are familiar with this design
- 14px vertical, 36px horizontal padding provides good click target

### Weaknesses
- May blend in with other green elements on page
- Expected CTR: 3.2%
