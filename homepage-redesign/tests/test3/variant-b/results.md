# Navigation Layout Test — Variant B (Sidebar)

Vertical sidebar navigation with fixed positioning at 240px width.

## Process

1. Created a fixed sidebar with vertical stacking of nav items
2. Used flex layout with sidebar + main content area
3. Same dark slate background and muted link colors as variant A

## Output

- `layout.html` — Full page layout with sidebar navigation
- `config.json` — Navigation configuration (type, position, width)

## Assessment

### Strengths
- More room for nav items and labels
- Can accommodate nested menu structures
- Always visible — no scrolling to find navigation

### Weaknesses
- Takes 240px of horizontal space from content
- May feel cramped on smaller screens
