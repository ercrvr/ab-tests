# Payment Form Layout — Single Page

All checkout fields presented on a single page using fieldset grouping.

## Process

1. Created a single form with two fieldsets: Shipping and Payment
2. Used bordered fieldsets with legend labels for visual grouping
3. Full-width "Place Order" button at the bottom

## Output

- `form.html` — Complete single-page checkout form

## Assessment

### Strengths
- Everything visible at once — user sees full scope
- Fewer clicks to complete
- 142-second average completion time (faster)

### Weaknesses
- Can feel overwhelming for users
- Higher cart abandonment rate (22.3%)
- Lower conversion rate (15.02%) compared to multi-step
