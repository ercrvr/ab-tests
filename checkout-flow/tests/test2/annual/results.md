# Pricing Display Format — Annual

Annual pricing display with savings badges and per-year billing information.

## Process

1. Created same two-column layout as monthly variant
2. Added green savings badges showing percentage saved vs monthly
3. Added "Billed $X/year" text below each price
4. Added "BEST VALUE" badge on Pro plan

## Output

- `pricing.html` — Two-column pricing card layout with annual pricing and savings badges

## Assessment

### Strengths
- Savings badges create urgency and perceived value
- "BEST VALUE" badge draws attention to higher-tier plan
- Yearly billing info provides transparency

### Weaknesses
- More visual complexity may slow decision-making
- Annual commitment may deter some users
